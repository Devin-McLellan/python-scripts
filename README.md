# Python Learning Journey

Practice scripts from the **100 Days of Code: The Complete Python Pro Bootcamp** on Udemy.

## Structure

```
python-scripts/
├── 01_Section/   - Printing & basics
├── 02_Section/   - Strings, data types & tip calculator
└── 03_Section/   - If, elif & else conditions
```

## Sections

### 01 - Printing & Basics
- Hello World and basic `print()` statements

### 02 - Strings, Data Types & Variables
- String indexing and concatenation
- Working with integers, floats and type conversion
- **Tip Calculator** — takes bill amount, tip %, and number of people to split the bill

### 03 - Control Flow & Conditionals
- `if`, `elif`, and `else` statements
- Nested conditions
- Rollercoaster ticket pricing based on height and age

## Setup

```bash
python -m venv .venv
.venv\Scripts\activate
python <script.py>
```
